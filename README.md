# Pico-8 Clone Demo

Small React + Vite demo: a Pico-8-like playable canvas game.

Controls
- Move: Arrow keys or WASD
- Z: Cycle player color
- X: Restart after game over

Score
- Score increases with seconds survived. If hit by a spawned object, game over.

Run (macOS, zsh)

```bash
cd /Users/winx/Desktop/Zhihao/2025/github-copilot-test
npm install
npm run dev
```

Open the URL printed by Vite (usually `http://localhost:5173`).
